# Overview

AI Resume Analyzer is a full-stack NLP web application that analyzes a candidate's resume and extracts:

Name, Email & Phone

Education & Experience

Technical Skills

Other Keywords

Resume Match Score

The system helps candidates understand how well their resume matches job requirements.
